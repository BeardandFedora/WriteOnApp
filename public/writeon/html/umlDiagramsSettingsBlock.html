Creates UML diagrams from plain text description. Note that the Markdown Extra extension has to be enabled with the GFM fenced code blocks option for this to work - and it's awesome when it does.

Sequence diagrams:

More info
```sequence
Fedora->Beard: Hello Beard, how are you?
Beard-->Fedora: I am good thanks!
```

Flow charts:

More info
```flow
st=>start: Start
e=>end
op=>operation: My Operation
cond=>condition: Yes or No?
st->op->cond
cond(yes)->e
cond(no)->op
```
